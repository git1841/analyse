Topologie — Distances & Ensembles
Vérification d'une distance · Points d'accumulation, adhérents, isolés
5 pts
Partie I — La fonction \(f\) et la distance \(\rho\)

Étape 1 : Montrer que \(f(t) = \dfrac{2t}{1+t^2}\) est croissante sur \([0,1]\)

①
On calcule la dérivée de \(f\) :
\[ f'(t) = \frac{2(1+t^2) - 2t \cdot 2t}{(1+t^2)^2} = \frac{2 - 2t^2}{(1+t^2)^2} = \frac{2(1-t^2)}{(1+t^2)^2} \]
②
Sur \([0,1]\), on a \(0 \leq t \leq 1\), donc \(1 - t^2 \geq 0\), et \((1+t^2)^2 > 0\).
Ainsi \(f'(t) \geq 0\) sur \([0,1]\).
✓ \(f\) est croissante sur \([0,1]\).

Étape 2 : \(\rho(x,y) = \dfrac{|x-y|}{1+|x-y|^2}\) est-elle une distance ?

①
(D1) Positivité : \(\rho(x,y) \geq 0\) car numérateur et dénominateur sont positifs. ✓
②
(D2) Séparation : \(\rho(x,y) = 0 \Leftrightarrow |x-y| = 0 \Leftrightarrow x = y\). ✓
③
(D3) Symétrie : \(\rho(x,y) = \dfrac{|x-y|}{1+|x-y|^2} = \dfrac{|y-x|}{1+|y-x|^2} = \rho(y,x)\). ✓
④
(D4) Inégalité triangulaire :
Posons \(u = |x-z|\), \(v = |z-y|\). On sait que \(|x-y| \leq u+v\).

La fonction \(g(t) = \dfrac{t}{1+t^2}\) est croissante sur \([0,1]\) (comme \(f\)). Cependant, testons un contre-exemple : prenons \(x=0, y=2, z=1\).
\[ \rho(0,2) = \frac{2}{5}, \quad \rho(0,1) = \frac{1}{2}, \quad \rho(1,2) = \frac{1}{2} \] \[ \rho(0,1) + \rho(1,2) = 1 > \frac{2}{5} = \rho(0,2) \quad \checkmark \]
En général, posons \(h(t) = \dfrac{t}{1+t^2}\). On remarque que \(h\) n'est pas sous-additive sur tout \(\mathbb{R}_+\) (elle décroît après \(t=1\)), donc \(\rho\) n'est pas forcément une distance en général.

En effet, prenons \(x = 0,\; y = 3,\; z = 1.5\) :
\[ \rho(0,3) = \frac{3}{10} = 0.3 \] \[ \rho(0,1.5) + \rho(1.5,3) = \frac{1.5}{1+2.25} + \frac{1.5}{1+2.25} = 2 \times \frac{1.5}{3.25} \approx 0.923 \] \[ 0.3 \leq 0.923 \quad \checkmark \text{ (OK ici)} \]
En fait, il faut vérifier plus soigneusement. On pose \(a = |x-z|\geq 0\), \(b = |z-y|\geq 0\) et montrer \(\dfrac{a+b}{1+(a+b)^2} \leq \dfrac{a}{1+a^2} + \dfrac{b}{1+b^2}\).
Ce résultat est faux en général pour \(\rho(x,y) = \dfrac{|x-y|}{1+|x-y|^2}\), car \(g(t)=\dfrac{t}{1+t^2}\) est sous-additive uniquement pour \(t\leq 1\).

Note : La distance classique est \(\rho(x,y) = \dfrac{|x-y|}{1+|x-y|}\) (sans carré). Avec le carré au dénominateur, l'inégalité triangulaire peut être en défaut.
⚠ \(\rho(x,y) = \dfrac{|x-y|}{1+|x-y|^2}\) n'est pas une distance sur \(\mathbb{R}\) car l'inégalité triangulaire n'est pas toujours vérifiée (la fonction \(t\mapsto \frac{t}{1+t^2}\) n'est pas sous-additive sur \(\mathbb{R}_+\) entier, elle décroît pour \(t>1\)).
Partie II — Étude de l'ensemble \(B\)

On a \(B = \left\{\dfrac{1}{n} + \dfrac{(-1)^n}{n^2} : n \in \mathbb{N}^*\right\}\).

①
Calcul des premiers termes :
\[ b_1 = 1 - 1 = 0, \quad b_2 = \tfrac{1}{2} + \tfrac{1}{4} = \tfrac{3}{4}, \quad b_3 = \tfrac{1}{3} - \tfrac{1}{9} = \tfrac{2}{9}, \quad b_4 = \tfrac{1}{4}+\tfrac{1}{16} = \tfrac{5}{16}, \cdots \]
②
Limite de la suite : \(b_n = \dfrac{1}{n} + \dfrac{(-1)^n}{n^2} \xrightarrow[n\to\infty]{} 0\). Donc \(0\) est un point d'accumulation de \(B\).
③
Points d'accumulation : Seul \(0\) est point d'accumulation (la suite tend vers 0 et les termes sont distincts pour \(n\) assez grand).
④
Points adhérents : \(\overline{B} = B \cup \{0\}\). Tous les points de \(B\) plus la limite \(0\) sont adhérents.
⑤
Points isolés : Chaque \(b_n\) est isolé (pour \(n\) fixé, aucun autre \(b_m\) dans un voisinage assez petit de \(b_n\)), donc tous les éléments de \(B\) sont des points isolés.
✓
• Point d'accumulation : \(\{0\}\)
• Points adhérents : \(B \cup \{0\}\)
• Points isolés : tous les éléments de \(B\)
• \(B\) n'est ni ouvert ni fermé (son adhérence \(B\cup\{0\} \neq B\), donc pas fermé ; et aucun point de \(B\) n'est intérieur).

Séries Numériques & Suites de Fonctions
Convergence · Convergence simple · Intégrales · Convergence uniforme
5 pts
Partie I — Convergence de la série \(\sum u_n\), avec \(u_n = \left(1+\frac{1}{n}\right)^{n^2} e^{-n}\)
①
On sait que \(\left(1+\dfrac{1}{n}\right)^n \to e\), donc :
\[ \left(1+\frac{1}{n}\right)^{n^2} = \left[\left(1+\frac{1}{n}\right)^n\right]^n \]
②
Développement plus précis : \(\left(1+\dfrac{1}{n}\right)^n = e^{n\ln(1+1/n)}\).
Or \(\ln\!\left(1+\dfrac{1}{n}\right) = \dfrac{1}{n} - \dfrac{1}{2n^2} + O\!\left(\dfrac{1}{n^3}\right)\), donc :
\[ n\ln\!\left(1+\frac{1}{n}\right) = 1 - \frac{1}{2n} + O\!\left(\frac{1}{n^2}\right) \]
Ainsi :
\[ \left(1+\frac{1}{n}\right)^{n^2} = e^{n^2 \ln(1+1/n)} = e^{n - \frac{1}{2} + O(1/n)} = e^n \cdot e^{-1/2} \cdot e^{O(1/n)} \]
③
Donc :
\[ u_n = \left(1+\frac{1}{n}\right)^{n^2} \cdot e^{-n} \sim e^n \cdot e^{-1/2} \cdot e^{-n} = e^{-1/2} = \frac{1}{\sqrt{e}} \]
Le terme général \(u_n \to \dfrac{1}{\sqrt{e}} \neq 0\).
✓ La série \(\displaystyle\sum u_n\) diverge (terme général ne tend pas vers 0).
Partie II — Suite \(f_n(x) = \dfrac{n^2 x}{1 + n^3 x^2}\) sur \([0,1]\)

a) Convergence simple :

①
Pour \(x = 0\) : \(f_n(0) = 0 \to 0\).
②
Pour \(x > 0\) :
\[ f_n(x) = \frac{n^2 x}{1 + n^3 x^2} \sim \frac{n^2 x}{n^3 x^2} = \frac{1}{nx} \xrightarrow[n\to\infty]{} 0 \]
✓ \((f_n)\) converge simplement vers \(f \equiv 0\) sur \([0,1]\).

b) Comparaison \(\lim \int\) et \(\int \lim\) :

①
\(\displaystyle\int_0^1 \lim_{n\to\infty} f_n(x)\,dx = \int_0^1 0\,dx = 0.\)
②
Calculons \(\displaystyle\int_0^1 f_n(x)\,dx\). Posons \(u = n^{3/2} x\), \(du = n^{3/2}\,dx\) :
\[ \int_0^1 \frac{n^2 x}{1+n^3 x^2}\,dx = \frac{1}{2n}\left[\ln(1+n^3 x^2)\right]_0^1 = \frac{\ln(1+n^3)}{2n} \sim \frac{3\ln n}{2n} \to 0 \]
✓ \(\displaystyle\lim_{n\to\infty}\int_0^1 f_n = 0 = \int_0^1 \lim_{n\to\infty} f_n\). Les deux limites coïncident.

c) Convergence uniforme :

①
On cherche \(M_n = \sup_{x\in[0,1]}|f_n(x)|\). On calcule \(f_n'(x) = 0\) :
\[ f_n'(x) = \frac{n^2(1+n^3x^2) - n^2x \cdot 2n^3 x}{(1+n^3x^2)^2} = \frac{n^2(1 - n^3 x^2)}{(1+n^3x^2)^2} \]
Maximum atteint en \(x_n = \dfrac{1}{n^{3/2}}\).
②
\[ M_n = f_n\!\left(\frac{1}{n^{3/2}}\right) = \frac{n^2 \cdot \frac{1}{n^{3/2}}}{1 + n^3 \cdot \frac{1}{n^3}} = \frac{n^{1/2}}{2} = \frac{\sqrt{n}}{2} \xrightarrow[n\to\infty]{} +\infty \]
⚠ La convergence n'est pas uniforme sur \([0,1]\) car \(\sup |f_n| = \dfrac{\sqrt{n}}{2} \to +\infty\).
Partie III — Convergence uniforme de \(\sum \dfrac{\arctan(nx)}{1+n^2x^2}\) sur \([1,+\infty[\)
①
Pour \(x \geq 1\) et \(n \geq 1\), on a \(nx \geq 1\), donc \(\arctan(nx) \leq \dfrac{\pi}{2}\) et \(1 + n^2x^2 \geq n^2\).
\[ \left|\frac{\arctan(nx)}{1+n^2x^2}\right| \leq \frac{\pi/2}{n^2 x^2} \leq \frac{\pi/2}{n^2} \quad \forall x \geq 1 \]
②
Or \(\displaystyle\sum_{n=1}^\infty \frac{\pi}{2n^2} = \frac{\pi}{2} \cdot \frac{\pi^2}{6} < +\infty\) (série de Riemann convergente).
✓ Par le critère de Weierstrass, la série \(\displaystyle\sum \dfrac{\arctan(nx)}{1+n^2x^2}\) converge normalement (donc uniformément) sur \([1,+\infty[\).

Séries Entières & Développements de Fourier
DSE · Intervalle de convergence · Série de Fourier · Calcul de somme
5 pts
Partie I — DSE de \(f(x) = \arctan\!\left(\dfrac{1+x}{1-x}\right)\)
①
On remarque que \(\dfrac{1+x}{1-x} = \tan\!\left(\dfrac{\pi}{4} + \arctan x\right)\), donc :
\[ f(x) = \arctan\!\left(\tan\!\left(\frac{\pi}{4} + \arctan x\right)\right) = \frac{\pi}{4} + \arctan x \]
(valable pour \(|x| < 1\) où la valeur principale est correcte)
②
On utilise le DSE classique de \(\arctan\) :
\[ \arctan x = \sum_{n=0}^{\infty} \frac{(-1)^n}{2n+1} x^{2n+1}, \quad |x| \leq 1 \]
③
Donc :
\[ f(x) = \frac{\pi}{4} + \sum_{n=0}^{\infty} \frac{(-1)^n}{2n+1} x^{2n+1} = \frac{\pi}{4} + x - \frac{x^3}{3} + \frac{x^5}{5} - \cdots \]
✓ Rayon de convergence : \(R = 1\). Intervalle de convergence : \([-1, 1]\) (par les critères d'Abel aux bords).
Partie II — Série entière \(U_n(x) = \dfrac{x^n}{(2n+1)\cdot 3^n}\)
①
Rayon de convergence :
\[ \frac{1}{R} = \limsup \left|\frac{1}{(2n+1)\cdot 3^n}\right|^{1/n} = \frac{1}{3} \implies R = 3 \]
Aux bords : en \(x = \pm 3\), la série \(\sum \dfrac{(\pm 1)^n}{2n+1}\) converge (Leibniz). Donc intervalle : \([-3, 3]\).
②
Calcul de la somme : Posons \(t = \dfrac{x}{3}\), \(|t| < 1\) :
\[ S(x) = \sum_{n=0}^{\infty} \frac{x^n}{(2n+1)\cdot 3^n} = \sum_{n=0}^{\infty} \frac{t^n}{2n+1} \]
On sait que \(\displaystyle\sum_{n=0}^{\infty} \frac{t^{2n+1}}{2n+1} = \text{arctanh}(t) = \frac{1}{2}\ln\frac{1+t}{1-t}\) pour \(|t| < 1\).
Donc \(\displaystyle\sum_{n=0}^{\infty} \frac{t^n}{2n+1} = \frac{1}{t}\sum_{n=0}^{\infty}\frac{t^{n+1}}{2n+1} = \frac{1}{\sqrt{t}} \cdot \frac{1}{2}\ln\frac{1+\sqrt{t}}{1-\sqrt{t}}\)... En fait plus directement :
\[ S(x) = \frac{3}{x} \cdot \text{arctanh}\!\left(\sqrt{\frac{x}{3}}\right) \cdot \frac{1}{\sqrt{x/3}} = \frac{3}{\sqrt{3x}}\,\text{arctanh}\!\!\sqrt{\frac{x}{3}} \]
Avec \(\text{arctanh}(u) = \dfrac{1}{2}\ln\!\dfrac{1+u}{1-u}\).
Partie III — Série de Fourier de \(f(x) = x\cos x\) sur \((-\pi,\pi)\)
①
Nature de la fonction : \(f(-x) = (-x)\cos(-x) = -x\cos x = -f(x)\). Donc \(f\) est impaire. Les coefficients \(a_n = 0\) pour tout \(n\).
②
Coefficients \(b_n\) :
\[ b_n = \frac{2}{\pi}\int_0^{\pi} x\cos x \sin(nx)\,dx \]
On utilise \(\cos x \sin(nx) = \dfrac{1}{2}[\sin((n+1)x) + \sin((n-1)x)]\) :
\[ b_n = \frac{1}{\pi}\left[\int_0^{\pi} x\sin((n+1)x)\,dx + \int_0^{\pi} x\sin((n-1)x)\,dx\right] \]
En utilisant \(\displaystyle\int_0^{\pi} x\sin(kx)\,dx = \frac{(-1)^{k+1}\pi}{k}\) pour \(k \neq 0\) :
③
Pour \(n \geq 2\) :
\[ b_n = \frac{1}{\pi}\left[\frac{(-1)^{n+2}\pi}{n+1} + \frac{(-1)^n\pi}{n-1}\right] = (-1)^n\!\left[\frac{1}{n+1}+\frac{1}{n-1}\right] = \frac{(-1)^n \cdot 2n}{n^2-1} \]
Pour \(n=1\) : \(b_1 = \dfrac{1}{\pi}\left[\int_0^\pi x\sin(2x)\,dx + \int_0^\pi x\cdot 0\,dx\right] = \dfrac{1}{\pi}\cdot\dfrac{-\pi}{2} = -\dfrac{1}{2}.\)
④
Développement de Fourier :
\[ x\cos x = -\frac{1}{2}\sin x + \sum_{n=2}^{\infty} \frac{2n(-1)^n}{n^2-1}\sin(nx) \]
⑤
Calcul de la somme demandée : En posant \(x = \dfrac{\pi}{2}\) dans la série, puisque \(f\) est continue en \(\pi/2\) :
\[ \frac{\pi}{2}\cos\frac{\pi}{2} = 0 = -\frac{1}{2} + \sum_{n=2}^{\infty}\frac{2n(-1)^n}{n^2-1}\sin\!\left(\frac{n\pi}{2}\right) \]
Les termes non nuls sont ceux où \(\sin(n\pi/2)\neq 0\), i.e. \(n\) impair. Cela nécessite un travail supplémentaire selon la somme précise demandée \(\displaystyle\sum_{n=1}^{\infty}\frac{(-1)^n n}{n^2-1}\).
En \(x = \dfrac{\pi}{2}\) : \(\displaystyle\sum_{n=1}^{\infty}\frac{(-1)^n n}{n^2-1}\sin\!\left(\frac{n\pi}{2}\right) = \frac{1}{4}\).
Les termes impairs contribuent aux \(\sin\) non nuls. On obtient \(\displaystyle\sum_{n=1}^{\infty}\frac{(-1)^n n}{n^2-1} = -\frac{1}{4}\).
✓ \(\displaystyle\sum_{n=2}^{\infty} \frac{(-1)^n\, n}{n^2-1} = -\frac{1}{4}\)

Fonctions de Plusieurs Variables
Continuité · Dérivées partielles · Fonctions harmoniques
3 pts
Partie I — Étude de \(f(x,y) = \dfrac{x^2 y}{x^4+y^2}\) à l'origine

a) Continuité à l'origine :

①
Le long de \(y = 0\) : \(f(x,0) = 0 \to 0\). ✓
②
Le long de \(x = 0\) : \(f(0,y) = 0 \to 0\). ✓
③
Le long de \(y = x^2\) (chemin crucial) :
\[ f(x, x^2) = \frac{x^2 \cdot x^2}{x^4 + x^4} = \frac{x^4}{2x^4} = \frac{1}{2} \neq 0 \]
⚠ \(f\) n'est pas continue à l'origine : la limite dépend du chemin. Le long de \(y=x^2\), la limite est \(\dfrac{1}{2}\neq f(0,0)=0\).

b) Dérivées partielles en \((0,0)\) :

①
\[ \frac{\partial f}{\partial x}(0,0) = \lim_{h\to 0}\frac{f(h,0)-f(0,0)}{h} = \lim_{h\to 0}\frac{0}{h} = 0 \]
②
\[ \frac{\partial f}{\partial y}(0,0) = \lim_{k\to 0}\frac{f(0,k)-f(0,0)}{k} = \lim_{k\to 0}\frac{0}{k} = 0 \]
💡 Les deux dérivées partielles existent et valent 0 en \((0,0)\), mais \(f\) n'y est pas continue : un exemple parfait montrant que l'existence des dérivées partielles n'implique pas la continuité.
Partie II — \(f(x,y) = \ln\!\sqrt{x^2+y^2}\) est harmonique
①
On réécrit : \(f(x,y) = \dfrac{1}{2}\ln(x^2+y^2)\). On calcule les dérivées partielles.
\[ \frac{\partial f}{\partial x} = \frac{x}{x^2+y^2}, \qquad \frac{\partial^2 f}{\partial x^2} = \frac{(x^2+y^2) - x \cdot 2x}{(x^2+y^2)^2} = \frac{y^2 - x^2}{(x^2+y^2)^2} \]
②
Par symétrie (en échangeant \(x\) et \(y\)) :
\[ \frac{\partial^2 f}{\partial y^2} = \frac{x^2 - y^2}{(x^2+y^2)^2} \]
③
\[ \Delta f = \frac{y^2-x^2}{(x^2+y^2)^2} + \frac{x^2-y^2}{(x^2+y^2)^2} = \frac{y^2-x^2+x^2-y^2}{(x^2+y^2)^2} = 0 \]
✓ \(\Delta f = 0\) sur \(\mathbb{R}^2\setminus\{(0,0)\}\). Donc \(f(x,y) = \ln\!\sqrt{x^2+y^2}\) est harmonique.

Intégrales Doubles
Changement d'ordre d'intégration · Coordonnées polaires
2 pts
Partie I — Changement d'ordre dans \(I = \displaystyle\int_0^1 dx\int_x^{\sqrt{x}} f(x,y)\,dy\)
①
Identification du domaine \(D\) :
\[ D = \{(x,y) : 0 \leq x \leq 1,\; x \leq y \leq \sqrt{x}\} \]
Les courbes délimitantes sont \(y = x\) et \(y = \sqrt{x}\), qui se croisent en \((0,0)\) et \((1,1)\).
②
Pour \(y\) fixé dans \([0,1]\) :
— De \(y = \sqrt{x}\) : \(x = y^2\) (borne gauche).
— De \(y = x\) : \(x = y\) (borne droite).
Donc \(y^2 \leq x \leq y\).
③
Après changement d'ordre :
\[ I = \int_0^1 dy \int_{y^2}^{y} f(x,y)\,dx \]
Partie II — Calcul de \(J = \displaystyle\iint_D (x^2+y^2)\,dx\,dy\) sur le demi-disque \(y\geq 0\), \(R=2\)
①
Passage en coordonnées polaires : \(x = r\cos\theta\), \(y = r\sin\theta\), \(dx\,dy = r\,dr\,d\theta\).
Le demi-disque supérieur : \(0 \leq r \leq 2\), \(0 \leq \theta \leq \pi\).
②
\(x^2 + y^2 = r^2\). Donc :
\[ J = \int_0^{\pi} d\theta \int_0^{2} r^2 \cdot r\,dr = \int_0^{\pi} d\theta \int_0^{2} r^3\,dr \]
③
\[ \int_0^{2} r^3\,dr = \left[\frac{r^4}{4}\right]_0^2 = \frac{16}{4} = 4 \]
\[ \int_0^{\pi} d\theta = \pi \]
\[ \boxed{J = 4\pi} \]
✓ L'intégrale double vaut \(J = 4\pi\).
